# 白玉楼製作所 放送部



## 需求

### 服务端

1. 根据挂载点查询的曲目查询、更新、生成翌日歌单

2. 接受客户端心跳



### 客户端

1. 全部曲目上传（扫描目录）、拉取翌日歌单（从服务端）

2. 向服务端发送心跳包



## 结构

+ ``/`` 页面

+ ``/info`` 活动挂载点列表

+ ``/playlist?m=&d=`` 根据挂载点获取当日歌单/翌日歌单

+ ``/update?m=`` 根据挂载点更新翌日歌单

+ ``/fulllist?m=`` 获取完整列表



## 上传结构

**``/fulllist?m=``**

曲目\t路径\n

曲目\t路径\n



**`/update?m=`**

曲目\n

曲目\n
